# Tensorbit Core
High-performance Rust library for Post-Training Tensor Decomposition. Optimize LLMs and Vision Transformers via Tucker/CP decomposition for NPU and mobile chips, purpose-built for Edge AI applications.

## Overview
Tensorbit Core is the foundational engine for **Tensorbit Labs**. It's a high-performance Rust engine specifically engineered for Post-Training Tensor Decomposition, providing a mathematically rigorous alternative to traditional bit-level quantization. By utilizing optimized Tucker, Canonical Polyadic (CP), and Singular Value Decomposition (SVD) algorithms, the library identifies and removes structural redundancies within large neural networks like LLMs and Vision Transformers. This project is also targetted towards usage in Edge AI Applications, and deploying AI systems on minimalistic and resource-constrained hardware.
